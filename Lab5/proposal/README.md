# **Capstone Project**

**การประเมินความเหมาะสมของพื้นที่สำหรับกิจกรรมทางน้ำในทะเลด้วยแบบจำลองเชิงพื้นที่ 
โดยใช้ตัวชี้วัดคุณภาพน้ำจากดาวเทียม Sentinel-2 บริเวณอ่าวไทยฝั่งตะวันออก**

**ส่วนที่ 1: Problem Statement**

ปัญหาคือ แม้ว่าทะเลจะเป็นพื้นที่ที่สามารถใช้ทำกิจกรรมได้ แต่ยังไม่มีการระบุเชิงพื้นที่ว่าบริเวณใดเหมาะสมจริงเมื่อพิจารณาทั้งคุณภาพน้ำและความปลอดภัยจากกิจกรรมมนุษย์ เช่น ท่าเรือและแท่นขุดเจาะน้ำมัน

งานนี้จึงใช้ตัวชี้วัดจากภาพดาวเทียม ได้แก่ Chlorophyll-a, Secchi Depth และ Trophic State Index (TSI) ร่วมกับระยะห่างจากแหล่งรบกวน เพื่อสร้างแผนที่ความเหมาะสมสำหรับกิจกรรมทางน้ำในพื้นที่อ่าวไทยฝั่งตะวันออกร่วมกับปัจจัยเชิงพื้นที่ที่มีผลต่อความเหมาะสมของกิจกรรมทางน้ำ

**▸ Specific — ระบุพื้นที่และประชากร/ระบบนิเวศที่ได้รับผลกระทบได้ชัดเจน**

ประชากรที่ต้องการทำกิจกรรมทางน้ำแบบ private บริเวณอ่าวไทยฝั่งตะวันออก

**▸ Measurable — มีตัวชี้วัดที่สามารถวัดได้ด้วย Remote Sensing หรือ GIS**

ใช้ Sentinel-2 จากคลังโค้ด 26_Sen2WaterQuality ใน GEE Code Editer

**▸ Relevant — เชื่อมโยงกับปัญหาจริงในบริบทไทยหรือภูมิภาค**

ทะเลอ่าวไทยมักพบปัญหาน้ำขุ่น เนื่องจากมีความลึกไม่มาก และมักเกิดปรากฏการณ์ algae bloom อยู่บ่อยครั้งทำให้คุณภาพน้ำทะเลแย่ได้

**ส่วนที่ 2: Research Questions**

มีพื้นที่ไหนบ้างในอ่าวไทยฝั่งตะวันออกที่เหมาะสมต่อการทำกิจกรรมทางน้ำ?

**▸ ข้อมูลที่ใช้ตอบ**
 * Code จาก repo 26_Sen2WaterQuality ได้ข้อมูลคุณภาพน้ำ โดย
      * Chlorophyll-a (ตัวชี้วัด eutrophication) คำนวณจาก NDCI (B5 - B4) / (B5 + B4)
      * Secchi Depth (ความใสของน้ำ) คำนวณจาก ln(B2 / B4)
      * Trophic State Index (TSI) คำนวณจาก 30.6 + 9.81 ln(chlorophyll-a)
 * ระยะห่างจากท่าเรือ
 * ระยะห่างจากแท่นขุดเจาะน้ำมัน


**▸ วิธีการวิเคราะห์** — Multi-Criteria Decision Analysis (MCDA)

**2. มีโครงการหรือ Paper ใดที่ทำสิ่งคล้ายกันแล้ว? ผลลัพธ์ของโครงการนี้จะต่างหรือเพิ่มเติมจากนั้นอย่างไร?**

**ตอบ** มีงานวิจัยจำนวนมากที่เกี่ยวข้อง เช่น
   งานวิจัยของ zheng et al. (2026) ที่เกี่ยวข้องในการใช้ sentinel-2 โดยงานวิจัยใช้ภาพถ่ายดาวเทียมเพื่อตรวจสอบค่าคลอโรฟิลด์ในระยะยาวทำให้เห็นว่าเซนเซอร์MSI บนดาวเทียมนี้มีประสิทธิภาพสูงในการประเมินคุณภาพน้ำในพื้นที่ขนาดใหญ่ และรวมไปถึงการใช้ดัชนีวัดความอุดมสมบูรณ์ของน้ำ ที่มีการใช้Chl-a เป็นตัวหลัก ในการประเมินสถานะเหมือนโครงการที่เราเลือกใช้
   มีความแตกต่างกันในด้านของพื้นที่ งานของzheng et al. (2026) ศึกษาใน Qiong Lake ซึ่งเป็นทะเลน้ำจืดในแผ่นดิน แต่โครงการของเราศึกษาในอ่าวไทย ซึ่งเป็นแหล่งน้ำเค็ม

**4. ใครคือ End-user จริงๆ ของผลลัพธ์? พวกเขาจะใช้แผนที่/โมเดลที่ได้ไปทำอะไรจริงๆ?**
   
   **ตอบ** (1)นักท่องเที่ยวและผู้ที่ทำกิจกรรมทางน้ำ - ใช้แผนที่เพื่อเลือกพื้นที่ที่มีคุณภาพน้ำดี มีความปลอดภัย และระดับความรบกวนต่ำ ซึ่งจะส่งผลต่อประสบการณ์การใช้งานโดยตรงต่อร่างกายและสุขภาพ

   (2) หน่วยงานภาครัฐและหน่วยงานแผนใช้พื้นที่ทะเล - สามารถนำผลลัพธ์ไปใช้ในการกำหนดเขตการใช้ประโยชน์พื้นที่ทางทะเล เช่น การจัดโซนพื้นที่อนุรักษ์ หรือกิจกรรมเฉพาะ เพื่อให้เกิดการใช้ทรัพยากรอย่างยั่งยืน
   
   (3)นักวิจัยและนักวิชาการด้านสิ่งแวดล้อมทางทะเล - สามารถนำข้อมูลและแบบจำลองไปเป็นข้อมูลพื้นฐานได้
